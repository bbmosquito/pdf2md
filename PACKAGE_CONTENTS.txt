【安装流程概览】

install.bat 自动执行以下 8 个步骤:

[1/8] 检查系统环境
  - 检测操作系统版本和架构
  - 检查内存大小
  - 验证管理员权限

[2/8] 检查并安装 Python 3.10+
  - 自动检测现有 Python
  - 下载并安装 Python 3.11.7 (如需要)
  - 配置环境变量

[3/8] 配置 pip 国内镜像源
  - 使用清华大学 TUNA 镜像
  - 加速依赖包下载

[4/8] 安装项目依赖
  - 安装 requirements.txt 中所有依赖
  - 使用国内镜像加速

[5/8] 安装 Docling OCR 引擎
  - 安装 Docling 核心包
  - 安装 OCR 插件

[6/8] 安装 PyTorch (GPU 加速)
  - 自动检测 GPU 类型
  - 安装对应版本的 PyTorch
  - 支持 NVIDIA CUDA 和 AMD ROCm

[7/8] 配置环境变量
  - 设置 PYTHONPATH
  - 配置 GPU 相关变量
  - 设置项目根目录

[8/8] 验证安装
  - 验证 Python 可用性
  - 验证 Docling 安装
  - 验证 PyTorch 安装
  - 验证所有依赖包

【使用示例】

基本转换:
  python pdf2md.py convert document.pdf

批量转换:
  python pdf2md.py batch ./pdfs

指定输出目录:
  python pdf2md.py convert document.pdf -o ./output

CPU 模式 (禁用 GPU):
  python pdf2md.py convert document.pdf --no-gpu

性能测试:
  python benchmark.py test.pdf

查看系统信息:
  python pdf2md.py info
